# ChatImage
A Minecraft mod which could show image in chat line  
在游戏聊天栏中显示你的图片  
## 用法
聊天框输入 `[chatimage,url=https://blog.kituin.fun/img/bg.png]`  
url=后面换成图片地址 目前还不支持本地
即可自动转换为图片
